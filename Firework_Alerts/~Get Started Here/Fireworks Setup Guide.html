Fireworks Alerts - WebM and Muxy.io Code







For use in any project, StreamLabels, or Muxy.io . Current version only has code support for Muxy.io. Stream labels will only support WebM files at the moment.

Muxy Setup

  1. Log into your Muxy.io account.
  2. Click the "Muxy Alert URL" in the main product folder or click here
  3. Click "copy and start editing"
  4. Make any changes to the "Media" tab (audio and text-to-speech)
  5. Load the alert package URL into your stream software
  6. Make sure width and height are set to 1920 wide by 1080 tall in stream software
  7. You may add sounds and text to speech as needed

StreamLabels Setup

  1. Log into your StreamLabs account
  2. Click the "Stream Labs Alert URL" in the download folder
  3. Click "Create Profile" button in StreamLabs - add profile name and description, then click "Create"
  4. Select "Use" to import the fireworks alert settings into your newly create StreamLabs profile
  5. Copy your Widget URL and paste it into your stream software via a browser source
  6. Make sure width and height are set to 1920 wide by 1080 tall in stream software
  7. You may add sounds and text to speech as needed

Muxy Setup (alternate)

  1. Log into your Muxy.io account.
  2. In the alerts section, create or edit a new alert package
  3. Select the alert type you'd like to edit
  4. Click the "Simple" button to change to "Advanced" settings
  5. Set "Animation" to "No Animation"
  6. Set "Display Time" to 13>
  7. Modify "Display Condition" as needed
  8. Under the "HTML" tab, completely remove all HTML
  9. Open the appropriate HTML file, located in the HTML folder for your product files in a program like notepad, copy all contents, and paste into the "HTML" area in Muxy
  10. Open the appropriate CSS file, located in the CSS folder for your product files in a program like notepad, copy all contents, and paste into the "HTML" area in Muxy
  11. In the "Media", remove the default "Image/Video" and replace with the WebM file of your choice. The short WebM videos will work fine here
  12. Make any changes to the "Media" tab (audio and text-to-speech)
  13. Load the alert package URL into your stream software
  14. Make sure width and height are set to 1920 wide by 1080 tall in stream software
